About Tamil Movie Directory

Tamil Movie Directory is a modern web application designed to showcase Tamil cinema with a clean, user-friendly interface. Our goal is to provide movie enthusiasts with an elegant platform to discover, explore, and organize their favorite Tamil films.

Comprehensive Database

Access detailed information about Tamil movies including cast, crew, ratings, and more.

Personal Favorites

Save your favorite movies and access them anytime with local storage persistence.

Smart Search

Find movies quickly with our powerful search and filtering capabilities.

Dark Mode

Enjoy comfortable viewing day or night with our elegant dark mode toggle.

Key Features

  • Responsive grid layout with beautiful movie cards
  • Multiple sorting options (Year, Rating, Alphabetical)
  • Genre-based filtering
  • Detailed movie pages with comprehensive information
  • Trailer viewing through YouTube integration
  • Streaming availability information
  • Share functionality for your favorite movies
  • Favorites management with local storage
  • Dark/light mode toggle
  • Smooth animations and transitions
  • Mobile-first responsive design

Technology Stack

HTML5 CSS3 JavaScript Tailwind CSS Font Awesome LocalStorage
